## 2D PONG GAME USING PYGAME MODULE
### Execute *main.py* to run the game
### Controls:
**Left paddle:**
1. *W - Up*
2. *S - Down*

**Right paddle:**
1. *Arrow_up - Up*
2. *Arrow_down - Down*

You must install pygame module first to run this code.
To install pygame:
1. Open Terminal
2. Type **pip install pygame**
